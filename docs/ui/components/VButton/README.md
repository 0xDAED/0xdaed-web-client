# VButton

Универсальная кнопка дизайн-системы.

## Ответственность

- Отрисовка кнопки
- Навигация по `to`
- Проброс кликов
- Поддержка tooltip
- Интеграция с DaisyUI

## Диаграммы

- `vbutton.class.puml` — структура компонента
- `vbutton.sequence.puml` — сценарии клика
- `vbutton.state.puml` — состояния кнопки

## Использование

См. `src/shared/ui/VButton`
